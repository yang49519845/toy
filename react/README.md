### 5.7
  
#### refs 使用场景（官方推荐）

* 管理焦点，文本选择或者媒体播放
* 触发动画
* 集成第三方DOM库
* dialog组件的 open 和 close 方法，最好传递isOpen属性，而不是使用ref

#### 不可以过度使用 refs

  refs 不建议过度使用，会破坏数据源(state, props)
